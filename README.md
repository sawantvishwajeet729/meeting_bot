## Run Locally

Clone the project

```bash
  git clone https://github.com/sawantvishwajeet729/meeting_bot.gitt
```

Go to the project directory

```bash
  cd meeting_bot
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the streamlit app

```bash
  streamlit run st_app.py
```
