## Deployment

To deploy this project run

```bash
  git clone https://github.com/sawantvishwajeet729/meeting_bot.git
```

Change directory to meeting_bot

```bash
    cd meeting_bot
```

Run the Streamlit app

```bash
    streamlit run st_app.py
```
